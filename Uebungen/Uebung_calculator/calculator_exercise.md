# Programmieraufgabe: Taschenrechner

## Aufgabenstellung
Entwickle ein prozedurales Python-Programm, das einen einfachen Taschenrechner simuliert. Das Programm soll den Benutzer über ein Menü durch verschiedene Rechenoperationen führen und die Ergebnisse speichern. Das Programm läuft so lange in einer Schleife, bis der Benutzer explizit die Option 'Beenden' auswählt.

Nutze das bereitgestellte Template oder starte eigenständig von Grund auf.

## Anforderungen
Implementiere für jede Rechenart eine eigene Funktion:

* add(a, b)
* subtract(a, b)
* multiply(a, b)
* divide(a, b)
  * Behandle den Sonderfall Division durch 0

## Ergebnissicherung
Führe nach Implementierung das Programm aus und beantworte dir folgende Fragen:
* Was kann das Programm?
* Warum stoppt das Programm nicht?
* Warum sieht das Ergebnis der Addition von 9 und 99 wie folgt aus `108.0`?


## Erweiterungen (optional)
* Ersetze den Platzhalter `op` im Verlauf durch das tatsächliche Operationszeichen `(+, -, *, /)`
* Füge eine Potenz- oder Wurzelfunktion hinzu
* Validiere die Benutzereingabe (Was passiert bei Buchstaben statt Zahlen?)
